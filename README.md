# Awesome NotePad App
An Implementation of Room Database with Google Recommended New Android Architecture with Kotlin for make a simple Note Pad app.

# Uses Technology/Kits
* MVVM
* Coroutines
* Stateflow
* Dagger Hilt
* UIState
